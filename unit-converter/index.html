사용 방법

• 변환 종류 선택: 상단 탭에서 원하는 변환 종류를 선택하세요.

• 값 입력: 변환하고 싶은 값을 입력하세요.

• 단위 선택: 변환할 단위와 변환될 단위를 선택하세요.

• 자동 변환: 단위를 선택하면 자동으로 변환 결과가 표시됩니다.

• 정확도: 소수점 6자리까지 표시되며, 불필요한 0은 자동으로 제거됩니다.
